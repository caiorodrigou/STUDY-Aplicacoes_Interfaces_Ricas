# Adapt: Portfólio Dinâmico de Arte Digital

## O que é?
O Adapt é um portfólio interativo de desenho digital diferente de uma galeria estática tradicional, o sistema se transforma organicamente para complementar a experiência visual do usuário se adaptando ao desenho mostrado.

O foco do projeto é sair da exibição estática de imagens e criar um ambiente digital onde a navegação, as transições e o layout reajam em tempo real ao conteúdo e às interações do usuário.

## Objetivos
* **Cores Dinâmicas:** O tema do site muda sozinho para combinar com a paleta de cores da arte que está sendo visualizada.
* **Navegação Fluida:** Troca de imagens e telas com animações suaves, sem a necessidade de recarregar a página.
* **Zoom Interativo:** Visualizador rápido que permite aproximar e explorar os detalhes dos desenhos de forma natural.
* **Filtros Animados:** Ao selecionar uma categoria (tag), os desenhos se reorganizam na tela de forma automática e animada.
